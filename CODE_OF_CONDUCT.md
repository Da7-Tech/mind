# Code of Conduct

Be respectful and constructive. This project follows the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/):

- Focus on the work: reproducible reports, measured claims, honest reviews.
- No harassment, personal attacks, or gatekeeping.
- Disagree with evidence, not volume.

Violations can be reported privately through GitHub's report feature or by
opening an issue asking for a private channel. The maintainer (Da7-Tech)
handles reports directly.
